# Mini Recipe Manager App

A small and clean recipe management application where users can create, store, edit, and organize recipes in one place.

## Overview

The Mini Recipe Manager allows users to:

- Add new recipes
- View existing recipes
- Delete recipes
- Organize and manage recipe collections
- Search and filter recipes

The app is designed to be simple, visually appealing, and easy to extend.

## Features

- Create recipes 
- Edit existing recipes
- Delete recipes
- Search recipes by title or ingredients
- Mark recipes as favorites (optional)

## Design Goals

- Minimal and modern interface
- Card-based layout for recipes
- Clear typography and spacing
- Simple navigation
